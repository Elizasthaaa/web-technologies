Temple Run

Temple Run is a popular endless running mobile game developed and published by Imangi Studios. In the game, the player controls a character who is running through a temple, collecting coins and avoiding obstacles. The character is chased by a group of demonic monkeys, and the goal of the game is to run as far as possible without being caught. Temple Run was initially released for iOS devices in 2011 and has since been made available on Android, Windows Phone, and other platforms. It has received positive reviews and has been downloaded over 1 billion times. The game has also spawned several sequels and spin-offs, including Temple Run 2, Temple Run Brave, and Temple Run Oz.
